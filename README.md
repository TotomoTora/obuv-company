# Проект для ООО "Обувь"

## Описание проекта
Информационная система для управления товарами обувного магазина.

## Функциональность
1. **Гостевая форма** - просмотр каталога товаров
2. **Форма менеджера** - управление товарами, поиск, фильтрация, сортировка

## Технологии
- C# / Windows Forms
- .NET Framework
- Git / GitHub

## Структура проекта
- `Program.cs` - точка входа
- `UserSelectionForm.cs` - форма выбора роли
- `GuestForm.cs` - форма для гостей (Task02)
- `ManagerForm.cs` - форма для менеджеров (Task03)

## Установка и запуск
1. Клонировать репозиторий
2. Открыть в Visual Studio
3. Собрать и запустить проект
